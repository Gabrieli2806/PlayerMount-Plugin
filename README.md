# PlayerMountPlugin

Allows players to ride each other, stay mounted during elytra flight, dismount by mount looking up and sneaking 3 times, toggle mounting and boss bar preferences, show mount health as a boss bar (optional), and attack other entities.

![Platform](https://img.shields.io/badge/platform-PaperMC%20%2F%20Bukkit-blue) ![API](https://img.shields.io/badge/api--version-1.21-green)

## Commands

| Command | Description | Permission |
|---------|-------------|------------|
| `/<command> <true|false>` | Toggle whether other players can mount you. | - |
| `/<command> <true|false>` | Toggle boss bar visibility when riding. | - |

## How to Use

1. Download the `.jar` from [Releases](https://github.com/Gabrieli2806/PlayerMount-Plugin/releases).
2. Place it inside your server's `plugins/` folder.
3. Restart or reload the server.
4. Use the commands listed above in-game or from the console.
